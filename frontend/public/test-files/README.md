## File Sources
- GrabCAD: https://grabcad.com/library (filter by STEP)
- TraceParts: https://www.traceparts.com/en
- McMaster-Carr: https://www.mcmaster.com (has STEP downloads)

## How to Add Files
1. Download STEP (.stp or .step) files
2. Place in this directory
3. Name them descriptively (e.g., simple_bracket.stp)
4. Keep under 10MB for fast testing

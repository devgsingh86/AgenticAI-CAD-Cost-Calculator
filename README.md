# AgenticAI CAD Cost Calculator

Enterprise-grade agentic AI system for automated CAD-to-cost estimation with Design for Manufacturing (DfM) optimization.

## Project Status
**Phase:** 1 - Planning & Requirements Definition  
**Started:** January 19, 2026  
**Tech Stack:** Next.js 15, OpenCascade.js (Wasm), LangGraph, Werk24, Pinecone

## Objectives
- Automate CAD file analysis (STEP/STL) → cost estimates in <5 seconds
- Generate DfM suggestions achieving 15-20% cost reduction
- Integrate real-time supplier quotes (Xometry/Fictiv)
